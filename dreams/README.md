# Z.E.T.A. Dreams

This directory contains "dreams" generated by the Z.E.T.A. system during its idle cycles.
These dreams represent the system's internal processing, consolidation of memories, and creative exploration of concepts.

## Dream Types

- **code_idea**: Proposed code improvements or new features.
- **code_fix**: Suggested fixes for potential bugs or issues.
- **insight**: General insights or philosophical reflections on the system's state.
- **story**: Narrative generation used for creative testing and memory consolidation.

## Structure

Each file is named with a timestamp and the type of dream: `dream_YYYYMMDD_HHMMSS_type.txt`.
